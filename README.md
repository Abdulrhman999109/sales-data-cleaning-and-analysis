# sales-data-cleaning-and-analysis

##  Project Overview

This project focuses on cleaning and analyzing a sales dataset containing missing values, inconsistent formats, and data quality issues.

The objective was to transform the raw dataset into a clean and reliable version, then generate meaningful business insights.

---


---

##  Tools Used

- Python
- Pandas
- Matplotlib
- Jupyter Notebook

---

##  Data Cleaning Steps

- Standardized text columns (Category, Region)
- Converted date columns to datetime format
- Converted numeric columns to proper data types
- Handled missing values
- Dropped rows missing critical numeric fields
- Created delivery_days feature
- Classified delivery status (OnTime, Late, Unknown)

---

##  Analysis & Insights

- Delivery performance distribution
- Top 3 regions by order volume
- Delivery status visualization (Pie Chart)

---

##  Files Included

- `orders_with_issues.csv` → Original dataset with issues
- `clean_orders.csv` → Cleaned dataset
- `sales_data_analysis.ipynb` → Full cleaning and analysis workflow

---

##  Key Learning Outcomes

- Data cleaning techniques
- Handling missing values
- Feature engineering
- GroupBy operations
- Basic data visualization
- Generating business insights from raw data
